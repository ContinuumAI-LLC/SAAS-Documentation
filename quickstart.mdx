## Get started in three steps

Get up and running with the Continuum AI API and make your first request.

### Step 1: Get your API key

<AccordionGroup>
  <Accordion icon="key" title="Create your Continuum AI account">
    Sign up at the [Continuum AI dashboard](https://yourai.continuumai.app/) and navigate to **API Keys** in the sidebar. Click **Create API Key**, give it a name, and copy the key to a safe location.

    <Warning>Your API key is shown only once. Store it securely and never expose it in client-side code.</Warning>
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Set up your environment">
    Export your API key as an environment variable so you can use it in requests:

    ```bash
    export CONTINUUM_API_KEY="your-api-key-here"
    ```

    <Tip>Add this to your shell profile (`.bashrc`, `.zshrc`) so it persists across sessions.</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Make your first request

<AccordionGroup>
  <Accordion icon="code" title="Send a chat completion request">
    Use `curl` to send your first request to the Continuum AI API:

    ```bash
    curl -X POST https://api.continuumai.technology/v1/chat/completions \
      -H "Authorization: Bearer $CONTINUUM_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "continuum-ultra",
        "messages": [
          {"role": "system", "content": "You are a helpful assistant."},
          {"role": "user", "content": "Hello, Continuum AI!"}
        ]
      }'
    ```

    You should receive a JSON response with the model's completion.
  </Accordion>
  <Accordion icon="brands python" title="Use Python">
    Make a request using Python's `requests` library:

    ```python
    import requests

    response = requests.post(
        "https://api.continuumai.technology/v1/chat/completions",
        headers={
            "Authorization": "Bearer your-api-key-here",
            "Content-Type": "application/json"
        },
        json={
            "model": "continuum-ultra",
            "messages": [
                {"role": "system", "content": "You are a helpful assistant."},
                {"role": "user", "content": "Hello, Continuum AI!"}
            ]
        }
    )

    print(response.json()["choices"][0]["message"]["content"])
    ```
  </Accordion>
</AccordionGroup>

### Step 3: Explore models and features

<Accordion icon="rocket" title="Choose your models">
  Continuum AI offers a range of proprietary models optimized for different tasks:

  | Model | Best for |
  |-------|----------|
  | `continuum-ultra` | Flagship. Complex tasks, analysis, creative work |
  | `continuum-turbo` | Low-latency, real-time applications |
  | `continuum-pro` | Balanced performance and cost |
  | `continuum-lite` | Simple tasks, classification, high volume |
  | `continuum-reason` | Math, logic, multi-step reasoning |
  | `continuum-embed-3` | Text embeddings for search and RAG |
  | `continuum-vision-hd` | Image generation up to 2048x2048 |
  | `continuum-voice-hd` | High-definition text-to-speech |
  | `continuum-listen` | Speech-to-text transcription |
  | `continuum-shield` | Content moderation (free) |

  See the full [Models catalog](/api-reference/models) for pricing, context windows, and capabilities.
</Accordion>
