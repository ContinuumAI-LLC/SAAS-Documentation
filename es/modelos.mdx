## Modelos disponibles

Continuum AI ofrece una amplia gama de modelos optimizados para diferentes tareas. Desde generacion de texto conversacional hasta embeddings, vision, audio y moderacion de contenido.

<Info>
  Todos los modelos son accesibles a traves de la URL base `https://api.continuumai.technology/v1`. Usa el identificador del modelo en el campo `model` de tus solicitudes.
</Info>

## Catalogo de modelos

<Tabs>
  <Tab title="Chat">
    Modelos de lenguaje para generacion de texto, razonamiento y conversaciones.

    | Modelo | Descripcion | Contexto | Mejor para |
    |--------|-------------|----------|------------|
    | `continuum-ultra` | Modelo insignia de maxima capacidad | 128K tokens | Tareas complejas, razonamiento avanzado, analisis profundo |
    | `continuum-ultra-mini` | Version compacta de Ultra | 128K tokens | Balance entre rendimiento y costo |
    | `continuum-turbo` | Alta velocidad, respuestas rapidas | 64K tokens | Aplicaciones en tiempo real, chatbots |

## Endpoints de modelos

### Listar modelos disponibles

Obtiene la lista completa de modelos disponibles en tu cuenta.

<ParamField header="Authorization" type="string" required>
  Bearer token con tu API key. Ejemplo: `Bearer sk-proj-...`
</ParamField>

<CodeGroup>

```bash curl
curl https://api.continuumai.technology/v1/models \
  -H "Authorization: Bearer sk-proj-tu-api-key"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-proj-tu-api-key",
    base_url="https://api.continuumai.technology/v1"
)

models = client.models.list()
for model in models:
    print(model.id)
```

```javascript Node.js
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "sk-proj-tu-api-key",
  baseURL: "https://api.continuumai.technology/v1",
});

const models = await client.models.list();
for (const model of models.data) {
  console.log(model.id);
}
```

</CodeGroup>

**Respuesta:**

```json
{
  "data": [
    {
      "id": "continuum-ultra",
      "object": "model",
      "created": 1706745600,
      "owned_by": "continuum-ai"
    },
    {
      "id": "continuum-turbo",
      "object": "model",
      "created": 1706745600,
      "owned_by": "continuum-ai"
    }
  ],
  "object": "list",
  "status": 200
}
```

<ResponseField name="id" type="string">
  Identificador unico del modelo. Usa este valor en el campo `model` de tus solicitudes.
</ResponseField>

<ResponseField name="object" type="string">
  Siempre `"model"`.
</ResponseField>

<ResponseField name="created" type="integer">
  Marca de tiempo Unix de la fecha de creacion del modelo.
</ResponseField>

<ResponseField name="owned_by" type="string">
  Organizacion propietaria del modelo.
</ResponseField>

### Obtener detalles de un modelo

Recupera la informacion detallada de un modelo especifico.

<ParamField path="model" type="string" required>
  Identificador del modelo. Ejemplo: `continuum-ultra`
</ParamField>

<CodeGroup>

```bash curl
curl https://api.continuumai.technology/v1/models/continuum-ultra \
  -H "Authorization: Bearer sk-proj-tu-api-key"
```

```python Python
model = client.models.retrieve("continuum-ultra")
print(model)
```

```javascript Node.js
const model = await client.models.retrieve("continuum-ultra");
console.log(model);
```

</CodeGroup>

**Respuesta:**

```json
{
  "data": {
    "id": "continuum-ultra",
    "object": "model",
    "created": 1706745600,
    "owned_by": "continuum-ai"
  },
  "status": 200
}
```
